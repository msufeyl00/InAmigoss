## 🚀 How to Use

### 1. **Open the Webpage**
- Double-click `index.html` to open in your default browser
- Or right-click → Open with → Choose your preferred browser

### 2. **Customize Content**
- Edit `index.html` in any text editor
- Replace placeholder text with actual NGO information:
  - Organization name and tagline
  - Project details and statistics
  - Contact information
  - Social media links

### 3. **Modify Styling**
- All CSS is embedded in the `<style>` tag
- Colors can be changed by modifying the color values
- Font sizes can be adjusted in the CSS section

### 4. **Add Images**
- Replace emoji placeholders with actual images:
  - In project sections: `.project-image`
  - In about section: `.about-image`
  - In impact section: `.impact-image`

---

## 📝 Customization Guide

### To Change Colors:
```css
/* Replace these color values in the <style> section */
Primary Color: #667eea
Secondary Color: #764ba2
Text Color: #333

```

### To Update Contact Info:
Modify the footer section:
```html
<p>Email: your-email@inamigos.org</p>
<p>Phone: +1 (XXX) XXX-XXXX</p>
<p>Address: Your Address, City</p>
```
